# FirebaseChatApp

## 何を作るのか？
ルームを作り、ルームごとにチャットをすることができる。

## 何を学べるのか？
* FireStoreでのデータの送受信
* ルーム機能を持ったチャットの作り方
* JSON解析
* プロフィール画像の表示

## ライブラリのインストール方法
```
  pod 'Firebase/Auth'
  pod 'Firebase/Firestore'
  pod 'IQKeyboardManagerSwift'
  pod 'ViewAnimator'
  pod 'Firebase/Storage'
  pod 'SDWebImage', '~> 5.0'
```
